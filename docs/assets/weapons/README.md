# Weapon .tik Files

Reference copies of weapon TIKI definition files from Medal of Honor: Allied Assault (and expansions).

These files are kept here so that AI assistants and developers can read them to understand weapon parameters, animations, and behavior without needing the original game assets installed.

## Organization

Files can be organized by game:

- `mohaa/` - Medal of Honor: Allied Assault
- `spearhead/` - Spearhead expansion
- `breakthrough/` - Breakthrough expansion

## Note

This directory is maintained separately from upstream. When syncing with upstream, these files should not be affected since upstream does not contain this directory. If rebasing causes issues, re-apply the commit(s) that added these files.
